@[TOC](23组 二进制时钟)
# 概要
## 项目名称
  二进制时钟
## 项目简介
  首先设计时钟外壳，通过激光切割木板得到外壳的各个部件，然后组装外壳。焊接LED作为二进制时钟的显示部分，以8421BCD码显示。编写程序，实现每一秒变化一次并且时钟进位正确。
## 耗材清单
木板*2（自费）
LED*20（自费）
esp32*1
充电宝*1（做电源）
# 用法
```
1.使用arduino IDE，打开code.c，将代码烧录至esp32
```

```
2.对底板、顶板、侧面、正面和背面五个部分进行激光切割打印
底板一块、顶板一块、侧面两块、正面一块、背面一块
```

```
3.组装二进制时钟外壳
```

```
4.对二进制时钟LED灯进行焊接
```

```
5.用充电宝做外部电源，通电即可使用
```
